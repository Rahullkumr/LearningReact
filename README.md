2. Run the command: `npm create vite` and proceed.
3. Enter a project name.
4. Leave the package name as default.
5. Select the framework (React).
6. Select JavaScript.
7. Run: `cd projectName`.
8. Run: `npm install`.
9. Run: `npm run dev`.
